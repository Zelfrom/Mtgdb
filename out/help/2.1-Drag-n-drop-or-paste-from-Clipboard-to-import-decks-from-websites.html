2.1 Drag n drop or paste from Clipboard to import decks from websites

Drag-n-drop files or folders

The fastest way to import a lot of decks is drag-n-drop some files or directories from Windows Explorer.


Another (less flexible) way is to select multiple deck files in Load deck(s) from from file(s)

dialog.

Drag-n-drop from websites

The simplest and fastest way to import a bunch of cards into your collection or current deck is to simply select the text using mouse and then

  • Drag-n-drop it into Mtgdb.Gui or
  • Copy (Ctrl + C) and Paste (see details)

The screenshots below demostrate drag-n-drop approach.

NOTE: Some MTG webistes let you download a deck in *.txt format. Then you can open it as MTGO .txt deck. See for example this webpage <https://magic.wizards.com/en/articles/archive/top-decks/what-we- learned-world-championship-2017-10-12>_. If you try hard enough you will find a download button next to twitter and facebook buttons in Deck title.
