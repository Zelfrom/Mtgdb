Get started

Prerequisites

The program requires .NET Framework v4.0. Most people have it already installed, in case you don't download it here.

You may also need to apply the latest patch to it.

.NET Framework v4.0 itself requires Windows, the oldest supported one is Windows XP.

Extract Mtgdb.Gui.zip

Use 7-zip or any other .zip file manager supporting advanced compression algorithms.

If you try extracting with default Windows .zip file manager, you will face this:

Screenshot above courtesy of user parkoro who reported this issue.

Running Mtgdb.Gui

Run Mtgdb.Gui shortcut in the root of extracted .zip

Downloading Images

Press the red cloud button in top right corner of Mtgdb.Gui window. An update window will open where you will see buttons to download images.
