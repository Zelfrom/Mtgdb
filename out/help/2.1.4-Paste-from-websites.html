Below screnshots illustrate drag-n-drop process for popular MTG-related websites. Select deck text using your mouse, then

  • drag it from the browser window and drop it into Mtgdb.Gui, or
  • copy Ctrl + C in browser and paste Ctrl + V it in Mtgdb.Gui
